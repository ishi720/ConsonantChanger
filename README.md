# 概要

ひらがなを特定の母音の列に変換します。

## 例

|変換前|変換内容|変換後|
|-|-|-|
|こんにちは|ぱ行|ぽんぴぴぱ|  
|ありがとう|さ行|さしさそす|
|おやすみなさい|が行|ごがぐぎががぎ|

# 環境

- Go: 1.22.1

# セットアップ

- リポジトリのクローン

```bash
$ git clone git@github.com:ishi720/ConsonantLockLanguage.git
```

- モジュールのダウンロード

```bash
$ go mod tidy
```

- アプリケーションの実行

```bash
$ go run main.go
```

- ブラウザで確認

http://localhost:8080/
