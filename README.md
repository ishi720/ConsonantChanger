# 概要

ひらがなを指定した母音の行に変換します。

## 例

|変換前|変換内容|変換後|
|-|-|-|
|こんにちは|ぱ行|ぽんぴぴぱ|  
|ありがとう|さ行|さしさそす|
|おやすみなさい|が行|ごがぐぎががぎ|

# 環境

- Go: 1.23.2

# セットアップ

- リポジトリのクローン

```bash
$ git clone git@github.com:ishi720/ConsonantLockLanguage.git
```

- モジュールのダウンロード

```bash
$ go mod tidy
```

- アプリケーションの実行

```bash
$ go run main.go
```

- ブラウザで確認

http://localhost:8080/
